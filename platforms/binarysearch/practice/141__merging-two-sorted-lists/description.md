# Description

Given two lists of integers `a` and `b` sorted in ascending order, merge them into one large sorted list.

**Constraints**
- `0 ≤ n ≤ 200,000` where `n` is the length of `a`
- `0 ≤ m ≤ 200,000` where `m` is the length of `b`
