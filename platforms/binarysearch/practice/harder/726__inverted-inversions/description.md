# Description

You are given a list of integers `nums`. Return the number of quadruplets `(a, b, c, d)` that exist such that `a < b < c < d` and `nums[a] < nums[b] and nums[c] > nums[d]`.

Return the answer modulo `10 ** 9 + 7`.
