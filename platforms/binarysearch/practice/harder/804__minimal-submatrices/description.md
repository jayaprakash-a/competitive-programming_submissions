# Description

Given a two-dimensional integer `matrix` and an integer `k`, return a matrix containing minimum values of all `k` by `k` submatrices.

**Constraints**
- `1 ≤ n * m ≤ 100,000` where `n` and `m` are the number of rows and columns in `matrix`
