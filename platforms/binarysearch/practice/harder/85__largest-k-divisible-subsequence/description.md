# Description

Given a list of non-negative integers `nums`, and a positive integer `k`, find the maximum sum [subsequence]{data-definition=subsequence-list-integer} of `nums` such that the sum is divisible by `k`.

**Constraints**

- `1 ≤ k ≤ 10`.
