# Description

Given a list of integers `nums` and an integer `k`, return the maximum possible `i` where `nums[0] + nums[1] + ... + nums[i] ≤ k`. Return `-1` if no valid `i` exists.

**Constraints**

- `0 ≤ n ≤ 1,000` where `n` is the length of `nums`.
- `-1,000 ≤ nums[i] ≤ 1,000`
- `0 ≤ k ≤ 10 ** 9`
