# Description

Given a binary search tree `root`, and `k` return the `k`th (0-indexed) smallest value in `root`. It is guaranteed that the tree has at least `k + 1` nodes.

**Constraints**
- `k ≤ n ≤ 100,000` where `n` is the number of nodes in `root`
