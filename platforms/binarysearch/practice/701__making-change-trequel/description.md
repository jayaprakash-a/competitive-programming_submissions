# Description

Given a list of integers `denominations` and an integer `amount`, return the number of combinations there are that sum to `amount`. Mod the result by `10 ** 9 + 7`.

**Constraints**

- `n ≤ 1000` where `n` is the length of `denominations`.
- `amount ≤ 2,000`
