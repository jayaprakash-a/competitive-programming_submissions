# Description

Given a list of integers `denominations` and an integer `amount`, find the minimum number of coins needed to make `amount`.

Return `-1` if there's no way to make `amount`.

**Constraints**

- `n ≤ 10` where `n` is the length of `denominations`.
- `amount ≤ 500,000`.
