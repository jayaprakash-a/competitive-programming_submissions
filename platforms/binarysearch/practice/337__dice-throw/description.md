# Description

Given integers `n`, `faces`, and `total`, return the number of ways it is possible to throw `n` dice with `faces` faces each to get `total`.

Mod the result by `10 ** 9 + 7`.

**Constraints**
- `1 ≤ n, faces, total ≤ 100`
