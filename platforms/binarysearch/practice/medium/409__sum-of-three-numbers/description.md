# Description

Given a list of integers `nums` and an integer `k`, determine if there are three distinct [elements]{data-definition=element} in the list that add up to `k`.

**Constraints**

- `n ≤ 1,000` where `n` is the length of `nums`
- `k ≤ 1,000`
