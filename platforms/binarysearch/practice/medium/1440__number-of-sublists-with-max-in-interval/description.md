# Description

You are given a list of integers `nums` and integers `lo` and `hi`. Return the number of sublists `A` such that `lo ≤ max(A) ≤ hi`.

**Constraints**
- `0 ≤ n ≤ 100,000` where `n` is the length of `nums`
