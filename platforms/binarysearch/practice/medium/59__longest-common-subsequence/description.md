# Description

Given two strings `a` and `b`, return the length of their longest common [subsequence]{data-definition=subsequence-string}.

**Constraints**

- `n ≤ 1,000` where `n` is the length of `a`
- `m ≤ 1,000` where `m` is the length of `b`
