# Description

You are given a singly linked list `node` and an integer `k`. Swap the value of the `k`-th (0-indexed) node from the end with the `k`-th node from the beginning.

**Constraints**
- `1 ≤ n ≤ 100,000` where `n` is the number of nodes in `node`
- `0 ≤ k < n`
