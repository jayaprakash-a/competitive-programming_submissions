# Description

You are given a string `s` containing `"1"`s and `"0"`s. Return the number of substrings that contain only `"1"`s. Mod the result by `10 ** 9 + 7`.

**Constraints**
- `0 ≤ n ≤ 100,000` where `n` is the length of `s`
