# Description

You are given integers `n` and `k`. Given that `n` represents the number of games you will play, return the number of ways in which you win `k` or less games consecutively. Mod the result by `10 ** 9 + 7`.

**Constraints**
- `n ≤ 10,000`
- `k ≤ 10`
