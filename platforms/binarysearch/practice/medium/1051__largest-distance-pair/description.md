# Description

You are given a list of integers `a` and `b` of the same length. Return the maximum value for all `0 ≤ i < j < n`:

```
|a[i] - a[j]| + |b[i] - b[j]| + |i - j|
```

**Constraints**
- `n ≤ 100,000` where `n` is the length of `nums`
