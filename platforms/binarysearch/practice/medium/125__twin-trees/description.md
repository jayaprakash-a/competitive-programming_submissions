# Description

Given two binary trees, `root0` and `root1`, return whether their structure and values are equal.

**Constraints**
- `n ≤ 100,000` where `n` is the number of nodes in `root0`
- `m ≤ 100,000` where `m` is the number of nodes in `root1`
