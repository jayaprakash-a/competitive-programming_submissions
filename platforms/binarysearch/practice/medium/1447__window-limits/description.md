# Description

You are given a list of integers `nums` and integers `window` and `limit`. Return whether `abs(nums[i] - nums[j]) ≤ limit` for every `i, j` such that `abs(i - j) < window`.

**Constraints**
- `1 ≤ n ≤ 100,000` where `n` is the length of `nums`
- `1 ≤ window ≤ n`
- `0 ≤ limit < 2 ** 31`
