# Description

Consider a sequence of sorted integers whose prime factors only include `2`, `3` or `5`. The first few terms are `1, 2, 3, 4, 5, 6, 8, 9, 10`.

Given an integer `n`, return the `n`th (0-index) term of the sequence.

**Constraints**
- `0 ≤ n ≤ 100,000`
