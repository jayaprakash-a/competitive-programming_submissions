# Description

You are given a list of integers `nums` and an integer `k`. Return the number of pairs `i < j` such that `(nums[i] + nums[j]) % k = 0`

**Constraints**
- `n ≤ 100,000` where `n` is the length of `nums`
- `1 ≤ k ≤ 100`
