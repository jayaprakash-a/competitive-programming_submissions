# Description

Given a list of positive integers `nums` and an integer `k`, return the number of subsets in the list that sum up to `k`.

Mod the result by `10 ** 9 + 7`.

**Constraints**

- `n ≤ 300` where `n` is the length of `nums`.
- `k ≤ 300`.
