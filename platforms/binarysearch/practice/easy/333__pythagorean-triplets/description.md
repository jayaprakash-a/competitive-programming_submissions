# Description

Given a list of positive integers `nums`, return whether there exist integers `a`, `b`, and `c` such that `a ** 2 + b ** 2 = c ** 2`.

**Constraints**
- `0 ≤ n ≤ 1,000` where `n` is the length of `nums`
