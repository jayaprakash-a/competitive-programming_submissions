# Description

You are given a list of integers `nums`, and an integer `k`. Given that you must remove `k` elements in `nums`, return the minimum `max(nums) - min(nums)` value we can achieve.

**Constraints**
- `1 ≤ n ≤ 100,000` where `n` is the length of `nums`
- `k < n`
