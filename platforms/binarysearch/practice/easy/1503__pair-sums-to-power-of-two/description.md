# Description

You are given a list of integers `nums`. Return the number of pairs `i < j` such that `nums[i] + nums[j]` is equal to `2 ** k` for some `0 ≤ k`.

**Constraints**
- `0 ≤ n ≤ 100,000` where `n` is the length of `nums`
