# Description

Given two alphabet (can be lower and/or uppercase) strings `s0` and `s1`, determine if one is a rotation of the other.

**Constraints**
- `1 ≤ n ≤ 200,000` where `n` is the length of `s0`
- `1 ≤ m ≤ 200,000` where `m` is the length of `s1`
