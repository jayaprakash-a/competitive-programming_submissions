# Description

Given a list of integers `nums` and an integer `k`, return `true` if you can remove exactly one element from the list to make the average equal to exactly `k`.

**Constraints**
- `2 ≤ n ≤ 1,000` where `n` is length of `nums`
- `nums[i], k ≤ 1,000,000`
