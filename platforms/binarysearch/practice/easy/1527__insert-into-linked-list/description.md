# Description

You are given a singly linked list `head` as well as integers `pos` and `val`. Insert a new node with value `val` before index `pos` of `head`.

**Constraints**
- `1 ≤ n ≤ 100,000` where `n` is the number of nodes in `head`
- `0 ≤ pos ≤ n`
