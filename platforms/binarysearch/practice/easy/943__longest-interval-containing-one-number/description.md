# Description

You are given a list of unique integers `nums`. Return the size of the largest inclusive interval `[start, end]` such that it contains exactly one number in `nums`.

**Constraints**
- `1 ≤ n ≤ 100,000` where `n` is the length of `nums`
- `1 ≤ start ≤ end ≤ 100,000`
