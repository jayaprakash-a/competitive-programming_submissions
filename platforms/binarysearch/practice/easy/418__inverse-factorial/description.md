# Description

The factorial of a number `n` is defined as `n! = n * (n - 1) * (n - 2) * ... * 1`.

Given a positive integer `a`, return `n` such that `n! = a`. If there is no integer `n` that is a factorial, then return `-1`.

**Constraints**
- `0 < a < 2 ** 31`
