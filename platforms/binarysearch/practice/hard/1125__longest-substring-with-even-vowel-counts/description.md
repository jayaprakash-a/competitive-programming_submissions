# Description

Given a lowercase alphabet string `s`, return the length of the longest substring where each vowel occurs even number of times. A vowel is either `"a"`, `"e"`, `"i"`, `"o"` or `"u"`.

**Constraints**
- `0 ≤ n ≤ 100,000` where `n` is the length of `s`
