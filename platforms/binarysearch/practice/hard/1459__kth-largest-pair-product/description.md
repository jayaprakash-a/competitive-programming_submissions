# Description

You are given two lists of integers `a` and `b`, and an integer `k`. Return the `k`th (0-indexed) largest possible product of `a[i] * b[j]`.

**Constraints**
- `1 ≤ n ≤ 100,000` where `n` is the length of `a`
- `1 ≤ m ≤ 100,000` where `m` is the length of `b`
- `0 ≤ k ≤ 100,000`
- `0 ≤ k < n * m`
