# Description

Given a two-dimensional integer `matrix` consisting only of `1`s and `0`s, return the area of the largest rectangle containing only `1`s.

**Constraints**

- `0 ≤ n, m ≤ 250` where `n` and `m` are the number of rows and columns in `matrix`
