# Description

Given a list of integers `nums` and an integer `k`, return the maximum values of each sublist of length `k`.

**Constraints**

- `1 ≤ n ≤ 100,000` where `n` is the length of `nums`.
- `1 ≤ k ≤ 100,000`.
